# 🚀 Cloudflare Pages Setup - Quick Start\n\n## What You Need\n\n1. **Cloudflare Account** - Sign up at https://dash.cloudflare.com (free tier works)\n2. **GitHub Account** - With access to skyofashions/ai-news-factory\n3. **5 minutes** - Time to complete setup\n\n---\n\n## Step 1️⃣: Get Your Credentials\n\n### Get API Token\n\n1. Go to https://dash.cloudflare.com/profile/api-tokens\n2. Click **\"Create Token\"**\n3. Select **\"Edit Cloudflare Workers\"** template\n4. Click **\"Use template\"**\n5. Verify permissions:\n   - ✅ Account → Cloudflare Pages → Edit\n   - ✅ All accounts selected\n6. Click **\"Continue to summary\"**\n7. Click **\"Create Token\"**\n8. **Copy the token immediately** ⚠️ (you won't see it again)\n\n```\nYour API Token: [paste here after copying]\n```\n\n### Get Account ID\n\n1. Go to https://dash.cloudflare.com\n2. Look for **\"Account ID\"** in the right sidebar\n3. Click the copy icon next to it\n\n```\nYour Account ID: [paste here after copying]\n```\n\n---\n\n## Step 2️⃣: Add GitHub Secrets\n\n1. Go to: https://github.com/skyofashions/ai-news-factory/settings/secrets/actions\n2. Click **\"New repository secret\"**\n\n### Add Secret 1\n- **Name:** `CLOUDFLARE_API_TOKEN`\n- **Value:** [paste your API token]\n- Click **\"Add secret\"**\n\n### Add Secret 2\n- **Name:** `CLOUDFLARE_ACCOUNT_ID`\n- **Value:** [paste your Account ID]\n- Click **\"Add secret\"**\n\n✅ **Done!** GitHub now has permission to deploy\n\n---\n\n## Step 3️⃣: Deploy\n\nChoose ONE method:\n\n### Method A: Automatic (Recommended)\n\n```bash\ncd ai-news-factory\ngit add .\ngit commit -m \"Deploy to Cloudflare Pages\"\ngit push origin main\n```\n\n→ GitHub Actions will automatically build and deploy\n\n→ Check: https://github.com/skyofashions/ai-news-factory/actions\n\n### Method B: Manual via Cloudflare Pages\n\n1. Go to https://pages.cloudflare.com\n2. Click **\"Create a project\"** → **\"Connect to Git\"**\n3. Click **\"Connect GitHub\"** and authorize\n4. Select **\"skyofashions/ai-news-factory\"**\n5. Click **\"Begin setup\"**\n\n**Build Settings:**\n```\nFramework: Next.js\nBuild command: npm run build\nBuild output directory: out\nRoot directory: / (leave empty)\n```\n\n**Environment Variables:**\n```\nName: NEXT_PUBLIC_SITE_URL\nValue: https://ai-news-factory.pages.dev\n```\n\n6. Click **\"Save and Deploy\"**\n\n---\n\n## Step 4️⃣: Verify Deployment\n\n✅ **Check deployment status:**\n\n- **GitHub Actions:** https://github.com/skyofashions/ai-news-factory/actions\n- **Cloudflare Pages:** https://pages.cloudflare.com → Select ai-news-factory\n\n✅ **View your site:**\n\n- **Live URL:** https://ai-news-factory.pages.dev\n\n---\n\n## 🎉 You're Done!\n\nYour site is now live on Cloudflare Pages!\n\n### Next Steps (Optional)\n\n- [ ] Add custom domain (instead of .pages.dev)\n- [ ] Enable Cloudflare analytics\n- [ ] Set up email notifications\n- [ ] Add branch protection rules\n\n---\n\n## ⚠️ Troubleshooting\n\n### \"Build failed\"\n\n**Solution:** Check GitHub Actions logs:\nhttps://github.com/skyofashions/ai-news-factory/actions\n\nCommon fixes:\n- Ensure `npm run build` works locally\n- Check Node.js version is 18+\n- Verify all dependencies in package.json\n\n### \"Site not found\"\n\n**Solution:** Wait 1-2 minutes for deployment to complete\n\nThen refresh: https://ai-news-factory.pages.dev\n\n### \"API Token invalid\"\n\n**Solution:** Regenerate token and update GitHub secret:\n\n1. https://dash.cloudflare.com/profile/api-tokens\n2.
