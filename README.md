# AI News Factory\n\nA production-ready AI News website built with Next.js 15, TypeScript, Tailwind CSS, and deployed on Cloudflare Pages.\n\n## Features\n\n- ✅ **Next.js 15 App Router** - Modern React framework with server components\n- ✅ **TypeScript** - Full type safety\n- ✅ **Tailwind CSS** - Utility-first CSS framework\n- ✅ **Dark Mode** - System preference detection and manual toggle\n- ✅ **Responsive Design** - Mobile-first approach\n- ✅ **SEO Optimized** - Meta tags, sitemap, and robots.txt\n- ✅ **RSS Feed** - Full RSS feed generation\n- ✅ **Search** - Full-text search functionality\n- ✅ **Admin Dashboard** - CMS for managing articles\n- ✅ **Markdown Support** - Write articles in Markdown\n- ✅ **Multiple Categories** - AI, Crypto, Technology, Business, Breaking News\n- ✅ **GitHub Actions** - Automated deployment\n- ✅ **Cloudflare Pages** - Global edge deployment\n\n## Getting Started\n\n### Prerequisites\n\n- Node.js 18+\n- npm or yarn\n\n### Installation\n\n```bash\n# Clone the repository\ngit clone https://github.com/skyofashions/ai-news-factory.git\ncd ai-news-factory\n\n# Install dependencies\nnpm install\n\n# Start development server\nnpm run dev\n```\n\nOpen [http://localhost:3000](http://localhost:3000) to view the site.\n\n## Project Structure\n\n```\n├── app/                    # Next.js app directory\n│   ├── layout.tsx         # Root layout\n│   ├── page.tsx           # Home page\n│   ├── [category]/        # Category pages (dynamic)\n│   ├── article/           # Article detail pages\n│   ├── search/            # Search page\n│   ├── admin/             # Admin dashboard\n│   ├── feed/              # RSS feed\n│   ├── sitemap.ts         # XML sitemap\n│   └── robots.ts          # Robots.txt\n├── components/            # React components\n├── lib/                   # Utility functions\n├── content/               # Content files (articles)\n├── public/                # Static files\n├── .github/workflows/     # GitHub Actions\n└── wrangler.toml         # Cloudflare configuration\n```\n\n## Content Management\n\n### Adding Articles\n\n1. Navigate to `/admin`\n2. Click \"New Article\"\n3. Fill in the form:\n   - **Title** - Article headline\n   - **Excerpt** - Short summary\n   - **Category** - AI, Crypto, Technology, Business\n   - **Author** - Author name\n   - **Tags** - Comma-separated tags\n   - **Content** - Markdown content\n4. Click \"Save Article\"\n\nArticles are stored as Markdown files in `content/articles/`\n\n### Manual Article Creation\n\nCreate a file in `content/articles/{slug}.md`:\n\n```markdown\n---\nid: unique-id\ntitle: Article Title\nexcerpt: Brief summary\ncategory: ai\nauthor: Author Name\ndate: 2024-01-15T00:00:00Z\ntags: [\"tag1\", \"tag2\"]\n---\n\n# Article Content\n\nYour markdown content here...\n```\n\n## Categories\n\n- **AI** - Artificial Intelligence news\n- **Crypto** - Cryptocurrency and blockchain\n- **Technology** - General tech news\n- **Business** - Business and finance\n- **Breaking News** - Latest urgent stories\n\n## Deployment\n\n### Cloudflare Pages\n\n1. Connect your GitHub repository to Cloudflare Pages\n2. Set build settings:\n   - **Framework preset** - Next.js\n   - **Build command** - `npm run build`\n   - **Build output directory** - `out`\n\n3. Add environment variables in Cloudflare dashboard\n\n4. Push to main branch - automatic deployment starts\n\n### GitHub Actions\n\nThe project includes a GitHub Actions workflow (`.github/workflows/deploy.yml`) that automatically builds and deploys to Cloudflare Pages on push to main.\n\n## Environment Variables\n\nCreate a `.env.local` file for development:\n\n```\nNEXT_PUBLIC_SITE_URL=http://localhost:3000\n```\n\nFor Cloudflare Pages deployment, add these secrets in GitHub Settings → Secrets and variables → Actions:\n\n- `CLOUDFLARE_API_TOKEN` - Generate at https://dash.cloudflare.com/profile/api-tokens\n- `CLOUDFLARE_ACCOUNT_ID` - Find in Cloudflare dashboard\n\n## Building for Production\n\n```bash\n# Build the project\nnpm run build\n\n# Start production server\nnpm start\n\n# Preview build\nnpm run preview\n```\n\n## API Routes\n\n### RSS Feed\n- **URL** - `/feed`\n- **Format** - RSS 2.0 with full article content\n\n### Sitemap\n- **URL** - `/sitemap`\n- **Format** - XML\n- **Includes** - All articles and category pages\n\n### Robots\n- **URL** - `/robots.txt`\n- **Disallows** - `/admin`, `/api`\n\n## Performance\n\n- **Static Generation** - Pages are pre-rendered at build time\n- **Image Optimization** - Disabled for Cloudflare Pages compatibility\n- **Caching** - HTTP headers optimized for edge caching\n- **SEO** - Sitemap, robots.txt, structured data\n\n## Browser Support\n\n- Chrome (latest)\n- Firefox (latest)\n- Safari (latest)\n- Edge (latest)\n\n## License\n\nMIT License - feel free to use this project as a template.\n\n## Support\n\nFor issues and questions, please open an issue on GitHub.\n\n## Contributing\n\nContributions are welcome!
